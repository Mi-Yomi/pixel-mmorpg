# Pixel MMORPG

Компактный Vite + TypeScript canvas-прототип MMORPG: первая локация `Деревня Ольховка`, NPC Мирон и Аня, слизни, цепочка заданий, HUD, инвентарь, золото, HP, автосохранение в `localStorage` и сенсорные кнопки.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

Проект использует существующие PNG из `public/assets/generated/` и не требует Phaser или других игровых движков.
